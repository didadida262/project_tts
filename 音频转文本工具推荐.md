# 音频转文本工具推荐（免费且性价比高）

## 一、开源工具（推荐）

### 1. OpenAI Whisper ⭐⭐⭐⭐⭐

**特点：**
- 完全免费开源
- 支持99种语言（包括中文）
- 准确率高
- 可离线使用
- 支持命令行和Python API

**安装：**
```bash
pip install openai-whisper
```

**使用方法：**
```bash
# 基本使用
whisper audio.wav --language zh

# 指定输出格式
whisper audio.wav --language zh --output_format txt

# 批量处理
whisper *.wav --language zh
```

**Python代码：**
```python
import whisper

# 加载模型（首次使用会自动下载）
model = whisper.load_model("base")  # tiny, base, small, medium, large

# 转录音频
result = model.transcribe("audio.wav", language="zh")

# 获取文本
text = result["text"]
print(text)

# 保存到文件
with open("transcript.txt", "w", encoding="utf-8") as f:
    f.write(text)
```

**模型选择：**
- `tiny`: 最快，准确率较低
- `base`: 平衡速度和准确率（推荐）
- `small`: 更好的准确率
- `medium`: 高准确率
- `large`: 最高准确率，但较慢

**优点：**
- 完全免费
- 可离线使用
- 支持批量处理
- 准确率高

**缺点：**
- 需要一定计算资源（GPU推荐）
- 大模型下载时间较长

### 2. SpeechRecognition + Google API

**特点：**
- 免费额度充足
- 准确率高
- 支持多种引擎

**安装：**
```bash
pip install SpeechRecognition pydub
```

**使用方法：**
```python
import speech_recognition as sr

# 初始化识别器
r = sr.Recognizer()

# 读取音频文件
with sr.AudioFile("audio.wav") as source:
    audio = r.record(source)

# 使用Google Web Speech API（免费）
try:
    text = r.recognize_google(audio, language="zh-CN")
    print(text)
except sr.UnknownValueError:
    print("无法识别音频")
except sr.RequestError as e:
    print(f"API错误: {e}")
```

**优点：**
- 使用简单
- 免费额度充足
- 准确率高

**缺点：**
- 需要网络连接
- 有使用限制

## 三、推荐方案

### 方案1：本地处理（推荐用于训练数据准备）

**使用Whisper：**
- 完全免费
- 可批量处理
- 数据隐私安全
- 适合大量音频文件

**安装脚本：**
```bash
# 安装Whisper
pip install openai-whisper

# 安装ffmpeg（用于音频处理）
# Windows: 下载 https://ffmpeg.org/download.html
# Linux: sudo apt install ffmpeg
# Mac: brew install ffmpeg
```

**批量处理脚本：**
```python
import whisper
import os
from pathlib import Path

# 加载模型
model = whisper.load_model("base")

# 音频目录
audio_dir = "data/audio"
output_dir = "data/transcripts"

os.makedirs(output_dir, exist_ok=True)

# 批量处理
for audio_file in Path(audio_dir).glob("*.wav"):
    print(f"处理: {audio_file.name}")
    
    # 转录
    result = model.transcribe(str(audio_file), language="zh")
    
    # 保存文本
    text_file = Path(output_dir) / f"{audio_file.stem}.txt"
    with open(text_file, "w", encoding="utf-8") as f:
        f.write(result["text"])
    
    print(f"完成: {text_file}")

print("批量处理完成！")
```

## 五、针对训练数据准备的推荐

### 最佳选择：Whisper

**理由：**
1. **完全免费** - 无使用限制
2. **批量处理** - 可处理大量音频文件
3. **数据隐私** - 本地处理，不上传服务器
4. **准确率高** - 适合生成训练数据
5. **可编程** - 易于集成到数据处理流程

**使用流程：**
```bash
# 1. 安装
pip install openai-whisper

# 2. 批量转换
whisper data/audio/*.wav --language zh --output_dir data/transcripts

# 3. 整理为metadata.csv格式
python prepare_metadata.py
```

**prepare_metadata.py示例：**
```python
import os
from pathlib import Path

audio_dir = "data/audio"
transcript_dir = "data/transcripts"
output_file = "metadata.csv"

# 生成metadata.csv
with open(output_file, "w", encoding="utf-8") as f:
    f.write("audio_path|text\n")
    
    for audio_file in Path(audio_dir).glob("*.wav"):
        transcript_file = Path(transcript_dir) / f"{audio_file.stem}.txt"
        
        if transcript_file.exists():
            with open(transcript_file, "r", encoding="utf-8") as t:
                text = t.read().strip()
                # 相对路径
                audio_path = f"data/audio/{audio_file.name}"
                f.write(f"{audio_path}|{text}\n")

print(f"metadata.csv 已生成: {output_file}")
```

## 六、快速开始

### 使用Whisper快速转换

```bash
# 1. 安装
pip install openai-whisper

# 2. 转换单个文件
whisper your_audio.wav --language zh

# 3. 批量转换
whisper data/*.wav --language zh --output_dir transcripts

# 4. 指定模型大小（base推荐）
whisper audio.wav --model base --language zh
```

### Python脚本示例

```python
import whisper
import os

# 加载模型
model = whisper.load_model("base")

# 转换单个文件
result = model.transcribe("audio.wav", language="zh")
print(result["text"])

# 批量转换
audio_dir = "data/audio"
for file in os.listdir(audio_dir):
    if file.endswith(".wav"):
        result = model.transcribe(
            os.path.join(audio_dir, file),
            language="zh"
        )
        # 保存文本
        with open(f"transcripts/{file}.txt", "w", encoding="utf-8") as f:
            f.write(result["text"])
```
